# Master of SEO Actions

Show this list when the user types `/` inside an active `/seo-master` workflow.

- `/seo-master /project-start` - set goals, market, audience, competitors, baseline state, and success criteria.
- `/seo-master /technical-audit` - audit crawlability, indexation, architecture, performance, schema, canonicals, redirects, and SEO risks.
- `/seo-master /keyword-research` - build keyword clusters by intent, funnel stage, difficulty, opportunity, and content type.
- `/seo-master /content-brief` - create SERP-aware content briefs with outline, entities, FAQs, internal links, and optimization notes.
- `/seo-master /on-page` - optimize titles, metas, headings, copy, images, schema, internal links, and page-level intent match.
- `/seo-master /competitor-gap` - compare competitors across rankings, content depth, SERP features, topical coverage, and links.
- `/seo-master /schema-plan` - recommend structured data types, required properties, validation steps, and implementation risks.
- `/seo-master /group-complete` - update memory, update README, and name the next action.
