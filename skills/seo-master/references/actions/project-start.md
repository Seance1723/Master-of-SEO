# /seo-master /project-start

## Purpose

Start an SEO project with enough context to avoid generic recommendations.

## Inputs

- Website or product URL
- Business model and location scope
- Primary goals and conversion events
- Audience, ICP, and buying stage
- Main competitors
- Known constraints, CMS, stack, and access limits
- Existing analytics, Search Console, rank tracking, or crawl data

## Output

- Project brief
- SEO success metrics
- Initial risk list
- Data/access checklist
- First recommended next action

## Closeout

Update `project-memory.md`, then update `README.md` Current Status.
