# /seo-master /on-page

## Purpose

Improve an existing page’s relevance, clarity, crawl signals, and conversion support.

## Checklist

- Intent match and page type fit
- Title tag, meta description, H1, and headings
- Above-the-fold clarity
- Main copy depth and entity coverage
- Internal links and anchor text
- Image filenames, alt text, size, and placement
- Schema fit
- E-E-A-T signals where relevant
- Conversion path and next step

## Output

- Recommended edits
- Replacement metadata
- Heading structure
- Internal-link plan
- Schema notes
- QA checklist

## Closeout

Update `project-memory.md`, then update `README.md` Current Status.
