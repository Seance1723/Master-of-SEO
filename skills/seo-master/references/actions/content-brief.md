# /seo-master /content-brief

## Purpose

Produce a SERP-aware brief that a writer or agent can use to create useful, rank-capable content.

## Inputs

- Target keyword cluster
- Search intent
- Target audience
- Existing URL or new page type
- Competitor/SERP observations when available

## Output

- Working title and URL slug
- Search intent summary
- H1, meta title, meta description
- Outline with H2/H3 structure
- Required topics, entities, examples, and proof points
- FAQ candidates
- Internal links in and out
- Schema recommendation
- Differentiation angle
