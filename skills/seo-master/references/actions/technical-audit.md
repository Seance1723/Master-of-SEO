# /seo-master /technical-audit

## Purpose

Find technical SEO issues that affect crawlability, indexation, rendering, ranking signals, and user experience.

## Checklist

- Crawl access: robots.txt, sitemap, status codes, redirects, canonicals
- Indexation: noindex, canonical targets, duplicates, pagination, faceted URLs
- Architecture: depth, internal links, orphan pages, navigation, breadcrumbs
- Rendering: JS dependency, blocked assets, mobile parity
- Performance: Core Web Vitals, image weight, caching, scripts
- Structured data: type fit, required properties, validation errors
- International/local: hreflang, geo pages, NAP consistency when relevant
- Risk: migrations, thin pages, generated pages, crawl traps

## Output

- Prioritized issue table
- Evidence and affected URLs
- Impact, effort, owner, and validation method
- Next action

## Closeout

Update `project-memory.md`, then update `README.md` Current Status.
