# /seo-master /keyword-research

## Purpose

Create a keyword map that links demand, intent, difficulty, and content opportunities.

## Workflow

- Seed the market from products, services, competitors, customer language, and Search Console if available.
- Cluster by search intent, entity, funnel stage, and page type.
- Separate primary, secondary, long-tail, question, comparison, and local modifiers.
- Mark cannibalization risks and content gaps.
- Prioritize by relevance, intent match, business value, ranking difficulty, and SERP fit.

## Output

- Keyword cluster table
- Recommended target page for each cluster
- Priority score and rationale
- New content, refresh, merge, and no-action recommendations
