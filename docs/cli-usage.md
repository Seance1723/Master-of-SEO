# CLI Usage

```bash
seo-master "/seo-master audit-website"
seo-master "/seo-master technical-audit --html '<html></html>'"
seo-master "/seo-master keyword-research technical seo, ecommerce seo"
seo-master "/seo-master seo-qa"
```

Commands without enough provided input return `needs_input` instead of fetching live data.
