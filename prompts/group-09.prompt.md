# Group 9 Prompt: Media SEO

Use `/seo-master media-audit`, `/seo-master image-seo-audit`, or `/seo-master video-seo-audit` to audit provided media inputs.

Use only explicit HTML, image, video, Open Graph, schema, asset, and manual data. Do not crawl, fetch, OCR, inspect live files, or invent image/video metadata.

If no usable input is provided, return `needs_input` and ask for HTML, image data, video data, Open Graph data, schema, or assets.
