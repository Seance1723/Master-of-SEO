# Group 10 Prompt: Ecommerce SEO

Use `/seo-master ecommerce-seo`, `/seo-master ecommerce-audit`, `/seo-master product-seo-audit`, or `/seo-master category-seo-audit` to audit provided ecommerce inputs.

Use only explicit product, category, variant, filter, pagination, policy, merchant feed, internal link, HTML, and page data. Do not fetch live product, Merchant Center, price, stock, shipping, return, or review data.

If no usable input is provided, return `needs_input` and ask for ecommerce inputs.
